# MovieDB (JS1-SlutProjekt)
This project was made to utilize the TMDB API and learn the fundamentals of APIs.
You can use this project in order to search for your favorite movies or actors, alongside seeing the top 10 most popular and top rated movies.

You can also click on a movie to see more information about it.

# Dependancies
This project utilizes anime.js to animate in the title and subtitle on the front page.

# More information.
The code is neatly split in different halves.
All HTML code is strictly in index.html, all JS are strictly in the JS files, same goes for CSS.

main.js is a simple main script that loads and initializes the website.
api.js is a module that handles everything about API calls.
renderContent.js is responsible for rendering mostly everything on the main website.
utils.js is for small utilties and error handling.
